# Turnerings-workflow

Brug dette workflow for konsekvent turneringsscoring.

## 1. Opret turnering

- Indtast turneringsnavn.
- Vælg antal spil pr. kamp.
- Vælg kampformat (VP eller Carry-over IMP).
- Vælg datum-skema.

## 2. Valgfrit brugerdefineret datum

- Slå brugerdefineret skema til.
- Tilføj titel.
- Upload CSV/TXT eller indsæt rækker.
- Gem skema før turneringen oprettes.

## 3. Indtast spil

- Indtast spiloplysninger og gem hvert spil.
- Gennemgå løbende totaler i spil-tabellen.
- Tjek IMP og VP i toppen.

## 4. Afslut eller udvid

Ved afslutning kan popup bruges til at:

- Gå tilbage til turneringer
- Udvide turnering
- Gå tilbage og fortsætte gennemgang

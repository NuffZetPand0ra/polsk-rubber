# Scoring og datum

Appen beregner IMP fra forskellen mellem faktisk score og datum-score.

## Input der bruges

- Kontrakttrin og farve
- Spilfører
- Resultat (over/under stik)
- Zonsestilling
- Doble eller redoble
- N/S HCP

## Datum-opslag

1. Vælg aktivt datum-skema.
2. Find HCP-rækken.
3. Hvis eksakt HCP mangler, bruges nærmeste lavere række.
4. Vælg ikke-zonestillet eller zonestillet kolonne.

## IMP-forskel

- Diff = Faktisk score (NS) - Datum-score
- IMP udledes af diff via IMP-tabellen.

## Hvorfor skema-valg betyder noget

Forskellige skemaer repræsenterer forskellige klubstandarder, så samme spil kan give lidt forskellig datum og IMP.

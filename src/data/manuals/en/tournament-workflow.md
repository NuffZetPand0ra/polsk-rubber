# Tournament Workflow

Use this workflow for consistent event scoring.

## 1. Create Tournament

- Enter tournament name.
- Choose boards per match.
- Choose match format (VP or Carry-over IMP).
- Choose datum schema.

## 2. Optional Custom Datum

- Enable custom schema checkbox.
- Add a title.
- Upload CSV/TXT or paste rows.
- Save custom schema before creating tournament.

## 3. Enter Boards

- Enter board details and submit each board.
- Review running totals in the board table.
- Check IMP and VP at the top.

## 4. Finish or Extend

At completion, use popup actions to:

- Return to tournaments
- Extend tournament
- Go back and continue reviewing

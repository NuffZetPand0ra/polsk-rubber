# Getting Started

Polsk Rubber lets you score board-by-board and compare actual score versus an HCP datum.

## Quick Flow

1. Create a tournament.
2. Select match format and datum schema.
3. Enter board results in Board Entry.
4. Track IMP and VP totals live.

## Modes

- Tournament mode: persistent board history and totals.
- Just Play: fast one-off entry.

## Datum Schemas

- Modern
- Polsk Rubber
- Classic
- Custom (uploaded/pasted at tournament creation)

## Tips

- Keep N/S HCP within 0-40.
- Use schema preview before starting a tournament.
- For custom schema, keep columns as: HCP, non vul, vul.
